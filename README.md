# SPARSH-CARE Website

SPARSH-CARE is a handmade soap and personal care products website developed as a web development project.

## Features

* Home Page
* Product Categories

  * Body Care
  * Face Care
  * Hair Care
  * Candles
* Product Showcase
* Customization Request Page
* Wholesale Inquiry Page
* Contact Page

## Technologies Used

* HTML5
* CSS3
